⏰ Digital Clock
----------------

A simple Digital Clock GUI built with Python (Tkinter).
It displays the current time in either 12-hour or 24-hour format,
with a clean and modern interface.

🚀 Features
-----------
- 🕒 Real-time clock display
- 🔄 Toggle between 12-hour and 24-hour formats
- 🎨 Stylish interface with Tkinter GUI
- ⏱️ Updates automatically every second

🛠️ Installation
----------------
1. Clone the repository:
   git clone https://github.com/your-username/DigitalClock.git
   cd DigitalClock

2. Run the script:
   python DigiClock.py

✅ No extra dependencies required — only Python 3.

▶️ Usage
--------
- Launch the app with:
  python DigiClock.py

- Click the "12hr / 24hr" button to switch formats.
